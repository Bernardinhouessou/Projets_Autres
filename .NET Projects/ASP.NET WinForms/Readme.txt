Q:A QUOI SERT Windows Forms ?

R: -------------------------------------------------------------------
Windows Forms est le nom de l'interface graphique qui est incluse dans le framework .NET
, fournissant l'accès via du Managed code à l'API Windows. Depuis la sortie de .NET 3.0 en 2006
, Windows Forms est destiné à être remplacé par un autre système d'interface graphique
, Windows Presentation Foundation.
